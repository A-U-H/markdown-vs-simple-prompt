# Assets Directory

This directory is for storing images and other assets for the portfolio website.

## Recommended Images

1. **Profile Photo** - A professional headshot for the About section
2. **Project Screenshots** - Screenshots for each project card
3. **Favicon** - A small icon for the browser tab

## Image Guidelines

- Use optimized image formats (WebP, JPEG, PNG)
- Keep file sizes under 500KB for faster loading
- Use descriptive file names (e.g., `profile-photo.jpg`, `amazon-ppc-dashboard.png`)
